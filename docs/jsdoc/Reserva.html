Class: Reserva

Reserva(usuario, libro, fechaReserva, fechaDevolucion)

La clase Reserva representa la reserva de un libro por parte de un usuario, con información sobre el usuario, el libro, la fecha de reserva y la fecha de devolución.

Constructor

new Reserva(usuario, libro, fechaReserva, fechaDevolucion)

Constructor para crear una reserva
Parameters:
Name Type Description
usuario string El usuario que realizó la reserva.
libro string El parámetro "libro" representa el libro que se está reservando.
fechaReserva Date El parámetro "fechaReserva" representa la fecha en que el usuario reservó el libro.
fechaDevolucion Date El parámetro "fechaDevolucion" representa la fecha en que se espera que el libro sea devuelto o la fecha de vencimiento para devolver el libro.
