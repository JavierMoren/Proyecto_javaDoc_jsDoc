Class: Libro

Libro(titulo, autor, ISBN, estado)

La clase representa un libro con propiedades como título, autor, ISBN y estado.

Constructor

new Libro(titulo, autor, ISBN, estado)

Función constructora que crea objetos de libros.
Parameters:
Name Type Description
titulo string El parámetro "titulo" representa el título del libro.
autor string El parámetro "autor" representa el autor del libro.
ISBN string El ISBN (Número Internacional Normalizado del Libro) es un identificador único para los libros. Se utiliza para identificar una edición específica de un libro.
estado string El parámetro "estado" representa la condición o estado del libro. Puede usarse para indicar si el libro es nuevo, usado, dañado u otra condición relevante.
